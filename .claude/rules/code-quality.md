# Kod Kalite Kuralları

- Fonksiyonlar tek bir iş yapsın (Single Responsibility).
- Fonksiyon uzunluğu 40 satırı geçmesin — geçiyorsa parçala.
- Cyclomatic complexity 10'u geçmesin.
- Magic number kullanma — sabit olarak tanımla.
- Yorum satırları "ne" değil "neden" açıklasın.
- Dead code bırakma — kullanılmayan kodu sil.
